# Java App Challenge

## Project Setup
#### create a emrdb database and load the sql file dump.sql (not necessary)
```
mysql -u [user] -p emrdb < dump.sql
```
#### Build Maven
```
./mvnw 
```
#### Run Test
```
./mvnw test 
```

[postman collection url]("https://documenter.getpostman.com/view/5832701/UzBvH3zu")
